para iniciar en python-Django

1. descargamos python desde la pagina oficial y lo instalamos.
2. despues de instalar python con el siguiente comando instalamos en el enterno virtual
    "pip install virtualenv"
3. En una consola vamos hasta la caperta principal Prueba Y realizamos el siguiente comando
    "virtualenv env"
4. crea una carpeta llamada env vamos a la siguiente ruta en una consola de comando
    "env\Scripts\activate"
5. en la carpeta Prueba principal damos el siguiente comando
    "pip install -r requeriments.txt"
6. despues de eso damos el siguiente comando para levantar el servidor
    python manager.py runserver
